# Dictionaries

You should download the [CMU Pronouncing Dictionary](http://www.speech.cs.cmu.edu/cgi-bin/cmudict) and [CC-CEDICT](https://cc-cedict.org/wiki/) and place them here.

```
curl http://svn.code.sf.net/p/cmusphinx/code/trunk/cmudict/cmudict-0.7b -o cmudict-0.7b
curl https://www.mdbg.net/chinese/export/cedict/cedict_1_0_ts_utf-8_mdbg.txt.gz -o cedict_1_0_ts_utf-8_mdbg.txt.gz
gunzip cedict_1_0_ts_utf-8_mdbg.txt.gz
```
